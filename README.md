# WS CEP

---

## Descrição

Biblioteca desenvolvida em PHP para consulta de CEP dos correios através de um WebService

## Configuração

1) Adicionar a dependência no composer.json:

```c++
$ composer require netoaoh/ws-cep
```

2) Instalar a biblioteca:

```c++
$ composer install
```

## Requisitos

- Composer
- PHP 5.4 ou superior
